Games

Game of Trades

Game of Trades is a JavaScript trading game taking place in the Game of Thrones universe. Players travel to different destinations buying and selling products, all while the events of the show affect the gameplay.

Star Wars RPG

Despite the name, Star Wars RPG is more of a simple strategy game than an RPG. The player selects a character and then must battle all the remaining characters until either he or they are defeated, with the order in which the enemies are picked determining success and failure.

Apps

NYT Search and Save is a full stack web application, built using the MERN stack, which allows users to search for a topic for articles on the New York Times and save articles to a list they can re-visit later. Includes Regex date validation and custom lists that are passed through the Mongo database to different React components that do not have the parent-child relationship.

Recre-ntals

Recre-entals is a fully-fleshed eCommerce site where users can go to rent (or rent out) sports and recreational equipment. Like an AirBNB for sports.

Bamazon

Bamazon is a Node CLI app built with inquirer which provides different levels of accesss (Customer, Employee, Supervisor) with different options, all dealing with a simple inventory/sales database built with MySQL.

Note: Because this is a node application, the link is to a GitHub repository. You will need to clone or fork the repo and go through the necessary installation steps detailed within to run the application properly
